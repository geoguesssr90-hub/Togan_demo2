# 陶眼 (TŌGAN)

天草陶石のAI等級判定システムのデモ。長崎学生ビジネスプランコンテスト用。

## 技術構成
- Vite + React + TypeScript(既存コンポーネントは .jsx のまま)
- メイン画面: src/Togan_demo.jsx

## デザインの約束
- 配色は磁器の白 (#F7F6F1) と呉須の藍 (#23508F) を基調に維持する
- 等級は 特級/一級/二級/規格外 の4段階

## 現状の判定ロジック
本物のMLではなく、画像の輝度統計による簡易解析(白色度・斑点・均一性)

## 既知の制約と設計判断
- 輝度ベース解析では、石表面の凹凸の影と鉄分斑点を原理的に区別できない
  (→ 機械学習導入の根拠。デモ時は明るい拡散光・平らな面の撮影で回避する)
- 閾値・係数は test-images/ の実写4枚で較正済み(2026年7月)。
  現場データ入手後に再較正する
- 閾値・係数を変更したら `npm run analyze:images` で全実写画像の判定変化を検証する
- 合成サンプル(src/lib/sampleStone.js)は4等級すべてをデモで提示できるよう
  意図的に調整されている(高品位=特級 / 中品位=一級・二級の境界 / 低品位=規格外)。
  判定ロジック側を変えた場合はサンプル側の再較正も必要
- LINE内蔵ブラウザではカメラ起動不可(LINE側仕様)。
  共有時は URL に `?openExternalBrowser=1` を付ける運用で回避する
