# 陶眼 (TŌGAN)

天草陶石のAI等級判定システムのデモ。長崎学生ビジネスプランコンテスト用。

## 技術構成
- Vite + React + TypeScript(既存コンポーネントは .jsx のまま)
- メイン画面: src/Togan_demo.jsx

## デザインの約束
- 配色は磁器の白 (#F7F6F1) と呉須の藍 (#23508F) を基調に維持する
- 等級は 特級/一級/二級/規格外 の4段階

## 現状の判定ロジック
本物のMLではなく、画像の輝度統計による簡易解析(白色度・斑点・均一性)

## 開発ルール
- コミットメッセージは英語(Conventional Commits形式: feat/fix/refactor/...)
